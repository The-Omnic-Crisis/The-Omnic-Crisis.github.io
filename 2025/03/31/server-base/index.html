安装完成后，打开 vscode 左边栏出现的小电脑，输入 ssh -p port username@ip，其中 port 是你的端口号，username 是你的用户名 (比如 root)，ip 是服务器地址。这样可以成功进入到服务器。然后点击出现的 Open，就可以打开相应的文件夹了。
